循环结构

  • while循环
a = 0
while a<10: # 判断语句 若结果为True，则运行下面的内容；为False，则结束循环
  a+=1
  print(a)

如果想让其在中途某个点退出，可按如下格式写

a = 0
while a<10: # 判断语句 若结果为True，则运行下面的内容；为False，则结束循环
  a+=1
  print(a)
  if a == 5:
    break # 跳出距它最近的循环

continue 跳过当前循环：

a = 0
while a<10: # 判断语句 若结果为True，则运行下面的内容；为False，则结束循环
  a+=1
  if a == 5:
     continue; # 当a == 5时，跳过，即不运行此处后面的语句
  print(a)
  • for 循环
for i in range(10):# 执行10次
  # range(10)
  # 列表[0,1,2,...,8,9 ]
  # i 循环变量 ，依次遍历列表中的值
  print('我错了')

in 后面的内容保证时一个列表、元组、字符串都可以。
range(n)：[0 n) 整数

  • 循环里的 else
for i in range(10):# 执行10次
  if a == 5:
    break; 
  print('我错了') 
else
  print("循环正常结束了")

其中如果else中print("循环正常结束了")运行，则表示循环正常结束，若不运行，则代表循环不是正常结束，可能是由break退出的。对while循环也一样

字符串处理

  • 切片
a  = "my name is xxx"
b = a[1:5]
# 取的下标为1到5但不包含5的字符
# b：'y na'
c = a[:5]
# 不写开始下标，默认从0开始
c = a[1:]
# 不写结束下标，则一直截取到最后
c = a[:]
# 什么都不写，代表切取的是全部
c = a[1:5:2]
# 前两个数表示开始和结束，第三个数表示间隔（步长），不写默认为1
c = a[::-1]
# 前面两个什么也不写，第三个写-1，则表示将字符串反过来
c = [-5:-1]
# 表示从倒数第五个字符截取到倒数第一个字符，但不包括倒数第一个字符
  • 替换
a = "my name is xxx"
arr = a.replace("xxx","pig") # 用 "pig" 替换 "xxx"
  • 根据某个字符进行分割
a = "my name is xxx"
arr = a.split(" ")# 根据空格进行分割
# ["my","name","is","xxx"]
  • 字符串列表拼合成字符串
arr = ["my","name","is","xxx"]  # 字符串列表
string = "-".join(arr) # "-"表示用什么来拼接它，可以选择其它的字符。
#  "my-name-is-xxx"
  • 转换大小写
a = "hello".capitalize()# # 将字符串第一个字母转换成大写
a = "hello".title()     # 将字符串每个单词首字母转换成大写
a = "hello".lower()     # 将字符串中大写转小写
a = "hello".upper()     # 将字符串中小写转大写
  • 删除空白字符
b = "     hello".lstrip()# 删除字符串左侧空白字符
b = "hello     ".rstrip()# 删除字符串右侧空白字符
b = "    hello  ".strip()# 删除字符串两侧空白字符

列表

用下标来获取字符串或列表里的字符或数据

  • 一维列表
string = "app"
s = string[0] # 'a'

arr = [2 3 1 7]
a = arr[1] # 3
  • 二维列表
arr = [[1,2,3],[4,5,6],[7,8,9]]
a = arr[1] # [4,5,6]
b = a[1][2] # 6
# 第一个下标表示拿到的子列表，第二个下标是子列表里面的数

列表是可以无限嵌套的，遍历高维列表可以采用嵌套的循环。
列表可以为所欲为，可以放任何你想要的数据和数据类型

b = 12
a = [1,False,''happy",b,[1,2,3]]
if 1 in a:
 print('...')# 如果1在a里面，则输出'...'
if 1 not in a:
 print('...'))# 如果1不在a里面，则输出'...'
  • 添加数据
b = 12
a = [1,False,''happy",b,[1,2,3]]
# 在a的末尾添加一个元素
a.apprnd("abc")
# 在任意位置插入元素
a.insert(2,'t') # 在下表为2的元素的前面插入一个元素't'
  • 删除
b = 12
a = [1,False,''happy",b,[1,2,3]]
a.pop(3) # 删除下标为3的元素
a.remove("happy") # 直接匹配某个元素并删除
a.clear() # 清空列表 a,a变成一个空列表[]
del a # 彻底删除整个列表
  • 修改
b = 12
a = [1,False,''happy",b,[1,2,3]]
a[0] = 123 # 通过下标修改该下标对应的元素

注意，列表可以通过下标进行修改，字符串不可以，因为字符串是不可变数据类型

  • 复制
b = 12
a = [1,False,''happy",b,[1,2,3]]
b = a.copy() # 将a的值复制给b

注意 若使用b = a，可以达到同样的效果，但这样子的话，修改a中元素，b中元素也会相应改变，反之亦然。

  • 其他操作
a = [4,5,3,1,2 ]
# 对a排序
a.sort()  # 直接操作并修改了a
# 或
a = shorted(a) # 有返回值，会返回一个新的列表

元组

元组和列表的区别在于，元组不能被修改，但我们可以访问元组中的值，方法和列表一样。

a = (1,2,3)

集合

集合不同于前面几种的地方在于它的两个特性：

1、无序性
2、不重复性

a = {1,3,3,4}
print(a)
# 输出为 {1,3,4}

在输入重复元素的情况下，它会自动去掉重复的元素
在写的时候{1,3,3,4}看上去是有顺序的，但在逻辑上，是没有顺序的

字典

字典也是由大括号括起来的，但其中内容是由一个一个的键值对组成的
键表示冒号前的数据，值表示冒号后的数据，二者可以是任意的数据类型

a = {"name":"pig","age":20}
  • 键与值
a = {"name":"pig","age":20}
a.keys()   # 获得字典中的所有键
# ['name','age']
a.values() # 获得字典中的所有值
# ['pig',20]
a.items()  #返回键值对组成的元组组成的列表
# [('name','pig'),('age',20)]
  • 遍历字典
a = {"name":"pig","age":20}
for k,v in a.items():
  print(k) # 键
  print(v) # 值

注意 a={}表示空字典而非空集合，空集合定义如下，b=set()

  • 字典操作
d = ("name":"pig""age": 20)
# 查找
print(d["name"])
# 修改
d["name"] ="Fog"
# 删除
del d["age"]
# 清空
d.clear()
# 彻底删除
del d

函数

def 函数名(传入参数):
  函数主要实现的功能
  return a,b,c #返回a,b,c

函数必须在调用前就已有声明，一般来讲，将函数定义在主程序前面即可。
下面给出一个例子

def mysum(a,b): # 这里的 a b是形式参数，其作用域仅在函数内部
 c = a+b
 return c
x = 1
y = 2
a = mysum(x,y)   # 这里的 x y是实际参数 
print(a)
# 3
