pix2tex的安装及使用

置顶 | 发表于 2023-05-11 分类于 工具类 阅读次数： Valine：
本文字数： 2k 阅读时长 ≈ 2 分钟

pix2tex可获取数学公式的图像并返回相应的 LaTeX 代码
Pix2Text 期望成为 Mathpix 的免费开源 Python 替代工具，完成与 Mathpix 类似的功能。当前 Pix2Text 可识别截屏图片中的数学公式、英文、或者中文文字。

Anaconda 安装：https://www.jianshu.com/p/ef1ae10ba950
Pytorch 安装：https://pytorch.org/get-started/locally/
pix2tex：https://www.jianshu.com/p/ef1ae10ba950
Anaconda换源：https://blog.csdn.net/watermelon1123/article/details/88122020

基础环境安装

  • Anaconda下载：https://www.anaconda.com/products/individual

  • Anaconda安装：

    • 勾选 Alls Users
    • 勾选 Add Anaconda to the system PATH environment variable
    • 选择一下安装位置，注意，安装目录以及上级目录不能为中文或者有空格
      安装完成后在开始菜单展开Anaconda（64-bit） 文件夹，双击Anaconda Prompt，注意，不是Anaconda Poweeshell Prompt！！！（我们接下来的造作均在此终端进行）

  • 换源

    • 添加清华源

      conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
      conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
      conda config --set show_channel_urls yes

    • 另外为了保险起见，建议同时添加第三方conda源：

      conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/
      conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/msys2/
      conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/bioconda/
      conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/menpo/
      conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/

    • 删除默认源

      conda config --remove channels defaults

  • 升级conda
    conda update conda
    升级时弹出Proceed ([y]/n)?时，选择y，确认升级。

  • 升级pip
    python -m pip install --upgrade pip

  • 环境管理

    • 新建虚拟环境
      需要新建一个虚拟环境来进行接下来的操作。
      conda create -n ptex python=3.7.12
    • 激活环境
      activate ptex
      至此，我们成功将环境切换到了ptex

Pytorch安装

  • 在ptex环境下，在终端中输入
    pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu117
    安装完成即可。
  • 为了保证正常使用，建议在终端中输入（可以在之后出现了’cannot import name ‘functional’ from ‘torchvision.transforms’ ‘在进行此步）
    pip install --upgrade torchvision

pix2tex安装

输入pip install pix2tex[gui]，然后等待安装完成。
安装后命令行输入pix2tex_gui打开pix2tex的图形界面。
在第一次运行pix2tex_gui需要下载参数包，因此需要等待一段时间。
下载完成后，就可以使用啦。

pix2tex使用

  • 点击Anaconda Prompt进入终端，使用activate ptex切换环境，然后在终端中使用pix2tex_gui即可打开图形界面。
  • 点击snip(Alt+s)进行截图，也可使用其快捷键
  • 若不满意可以点击retry
  • 有时需要重新截取，目前测试，对一行公式识别较好，多行公式需要其通过大括号包裹，此时识别效果也较好。
